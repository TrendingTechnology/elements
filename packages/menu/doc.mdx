## Description

Menus display a list of actions or choices on a temporary card. They offer a condensed way to present multiple actions and actions that a user won't always need to see.

1. **DropdownMenu Surface** This is a card background that contains the menu. It's min-width is 240px and grows to include the length of the longest item.

2. **DropdownItem**. This is a single action or choice contained by the menu.

3. **More Action**. Menus can open other menus. This should not be used often, it is an affordance of the menu list however.
