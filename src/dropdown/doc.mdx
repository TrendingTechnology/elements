## Installation

```
import {
  DropdownMenu,
  DropdownItem
} from "@rent_avail/elements/dropdown"
```

## Description

Dropdowns display a list of actions or choices on a temporary card. They offer a condensed way to present multiple actions and actions that a user won't always need to see.

1. **DropdownMenu Surface** This is a card background that contains the menu. It's min-width is 228px and grows to include the length of the longest item.

2. **DropdownItem**. This is a single action or choice contained by the menu.

3. **More Action**. Dropdown menus can open other dropdown menus. This should not be used often, it is an affordance of the menu list howeever.

## DropdownMenu Props

<Prop name="anchor" type="reference" required>
  A reference to a DOM node that the Dropdown Menu will appear against.
</Prop>
<Prop name="open" type="boolean" required>
  Determines whether to show the dropdown or not.
</Prop>
<Prop name="toggle" type="function" required>
  Function to toggle the state of open.
</Prop>
<Prop name="children" type="node" required>
  A set of <code>DropdownItems</code> that make up the menu actions.
</Prop>
