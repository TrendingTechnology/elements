# Grids

<Status status="production" />

## Installation

```js
import { Grid, Col } from "@rent_avail/elements/grid"
```

## Description

Grids are used to build consistent, responsive layouts. The `<Grid/>` component can be used as a row or an entire grid. They use css-grid under the hood and are absracted to give you as little headache as possible. The `<Col />` component should be the immediate child of a grid component. Columns have been redesigned from v1 to work with an array prop syntaxt. Each `<Col />` takes a `span` prop with a number of columns to span, by default the column with span all 12 columns.

## Anatomy

### Grid Props

<Prop name="column" type="number" defaultValue="12">
  Optional override on the number of columns the grid contains.
</Prop>

<Prop name="gap" type="string" defaultValue="2rem">
  Optional override on the gap inbetween columns and rows.
</Prop>

<Prop name="xsmBreak" type="string" defaultValue="40rem">
  Optional override for the extra small breakpoint.
</Prop>

<Prop name="smBreak" type="string" defaultValue="60rem">
  Optional override for the small breakpoint.
</Prop>

<Prop name="mdBreak" type="string" defaultValue="80rem">
  Optional override for the medium breakpoint.
</Prop>

<Prop name="lgBreak" type="string" defaultValue="100rem">
  Optional override for the large breakpoint.
</Prop>

<Prop name="xlgBreak" type="string" defaultValue="120rem">
  Optional override for the extra large breakpoint.
</Prop>

### Col Props

<Prop name="span" type="array" defaultValue={`[12]`}>
  The number of columns to span in a row, cannot exceed the columns allowed on parent in
  any one element.
</Prop>

<Prop name="offset" type="array" defaultValue={`["auto"]`}>
  Optional amount of spaces from left to offset the column.
</Prop>

## Usage

### Columns

<Playground column>
  {() => {
    const styles = {
      background: colors.green_500,
      height: "20rem",
      display: "flex",
      alignItems: "center",
      justifyContent: "center"
    }
    return (
      <Grid>
        <Col style={styles} span={[12, 6, 4, 3, 12]}>
          <p>1</p>
        </Col>
        <Col style={styles} span={[4, 6, 4, 3, 4]}>
          <p>2</p>
        </Col>
        <Col style={styles} span={[4, 6, 4, 3, 4]}>
          <p>3</p>
        </Col>
        <Col style={styles} span={[4, 6, 12, 3, 4]}>
          <p>4</p>
        </Col>
      </Grid>
    )
  }}
</Playground>

You can easily offset columns in the same manner as the `span` prop.

<Playground column>
  {() => {
    const styles = { background: colors.blue_500, height: "20rem" }
    return (
      <Grid>
        <Col style={styles} span={[6]} offset={[3]} />
      </Grid>
    )
  }}
</Playground>
