# Checkboxes

<Status status="production" />

## Installation

```javascript
import Checkbox from "@rent_avail/elements/checkbox"
```

## Description

Checkboxes are user selection tools when multiple options can be chosen at the same time. A user should be able to select one or more options from a set.

## Anatomy

### Props

<Prop name="checked" type="boolean" defaultValue="false">
  Boolean to determine the visual state of the checkbox.
</Prop>

## Usage

### Basic

<Playground>
  <Checkbox />
  <Checkbox checked />
  <Checkbox />
</Playground>

### With a label

A checkbox's children will render as the lable to the right of the element. Checkbox's default to inline and don't have any native padding or margin.

<Playground column>
  <h4>Which kinds of pets do you allow?</h4>
  <Checkbox>Allow Dogs</Checkbox>
  <Checkbox>Allow Cats</Checkbox>
  <Checkbox>Allow Parrots</Checkbox>
</Playground>
