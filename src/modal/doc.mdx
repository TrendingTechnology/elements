# Modals

<Status status="production" />

## Installation

```js
import Modal from "@rent_avail/elements/modal"
```

## Description

Modals are ways to ask a user for confirmation or to take action without losing context of the current screen. Modals should always be something that would best be done now, but don't make sense to navigate the user away from their current process for.

## Anatomy

### Props

<Prop name="open" type="boolean" defaultValue="false">
  Boolean that determines whether the modal is currently shown.
</Prop>
<Prop name="toggle" type="function" defaultValue="() => null">
  Callback function that should update the state of the open boolean.
</Prop>

## Usage

<Playground>
  {() => {
    const [open, setOpen] = useState(false)
    const handleOpen = e => setOpen(!open)
    return (
      <div>
        <Button onClick={handleOpen}>Toggle Modal</Button>
        <Modal open={open} toggle={handleOpen}>
          Hello World
        </Modal>
      </div>
    )
  }}
</Playground>
