# Radio Buttons

<Status status="production" />

## Installation

```js
import Radio, { RadioGroup } from "@rent_avail/elements/radio"
```

## Description

Radio buttons are used to select one option from a set. Use radio buttons when it's helpful for the user to see all the available choices when selecting.

## Radio Anatomy

### Props

<Prop name="checked" type="boolean">
  Controls the checked nature of the radio button to give the user feedback.
</Prop>

## RadioGroup Anatomy

### Props

<Prop name="inline" type="boolean">
  Controls whether the radio buttons render on the same line or on different rows.
</Prop>

## Usage

### Radio

<Playground>
  <Radio checked />
  <Radio />
</Playground>

### Radio Group

To easily group buttons into usable forms, use the `RadioGroup` component

<Playground column>
  <RadioGroup inline>
    <Radio name="user-type" />
    <Radio name="user-type" />
  </RadioGroup>
  <RadioGroup style={{ marginTop: "2rem" }}>
    <Radio name="payment-type" />
    <Radio name="payment-type" />
  </RadioGroup>
</Playground>
