# Menus

<Status status="beta" />

## Installation

```js
import { DropDown, DropDownInput } from "@rent_avail/elements/menu"
```

## Description

Menus display a list of actions on a temporary surface. They appear when users interact with a button, icon, or input. There are two types of menus, `DropDown` & `DropDownInput`.

## Anatomy

### DropDown Props

<Prop name="open" type="boolean" defaultValue="false">
  The boolean attribute that determines if the dropdown is open or not.
</Prop>
