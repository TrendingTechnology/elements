## Installation

```
import Avatar, { AvatarGroup } from "@rent_avail/elements/avatar"
```

## Description

An avatar component is used to display user information to another user. It can be used in a variety of sizes and spaces and is often clickable and opens more in-depth information about the user such as account or payment status.

### Props

<Prop name="small" type="boolean">
  Boolean to dictate the size and layout of an avatar. Small avatars should only be used
  in an <code>AvatarGroup</code>
</Prop>

<Prop name="initials" type="string">
  The intials of the user associated with the avatar.{" "}
  <strong>Only used in small avatars.</strong>
</Prop>

<Prop name="name" type="string">
  The name of the user associated with the avatar.{" "}
  <strong>Not used in small avatars.</strong>
</Prop>

<Prop name="photoUrl" type="string:url">
  <p>
    The profile photo of the user if there is one. Defaults to a solid color if no photo
    is present. <strong>Not used in small avatars.</strong>
  </p>
</Prop>

<Prop name="icon" type="ReactElement">
  <p>
    an icon to be displayed to the right of a users name.{" "}
    <strong>Not used in small avatars.</strong>
  </p>
</Prop>

## Usage

### Avatar

<Playground>
  <Avatar small initials="JW" />
  <Avatar
    name="Janet Wood"
    photoUrl="https://source.unsplash.com/collection/802865/48x48"
  />
  <Avatar
    name="Janet Wood"
    photoUrl="https://source.unsplash.com/collection/802865/48x48"
    icon={ExternalLink}
  />
</Playground>

### Avatar Group

Stacked avatars represent a group of people

<Playground>

</Playground>

## Extension

The avatar component passes all props to its underlying dom node.

```js
const CustomAvatar = styled(Avatar)`
  background: red;
  color: green;
`
```
