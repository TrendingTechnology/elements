# Container

<Status status="production" />

## Installation

```js
import Container from "@rent_avail/elements/container"
```

## Description

Containers are used to prevent the layout from becoming unusable on large scale screens. They have a max-width and automatically center to their parent.

## Anatomy

### Props

<Prop name="maxWidth" type="string" defaultValue="80rem">
  The default maximum width that a container will stretch to.
</Prop>

## Usage
