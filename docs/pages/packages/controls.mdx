## Description

This package contains a set of elements that allow a user to control state on a page.

## Button

```js
import { Button } from "@availkit/controls"
```

### Style Props

`color, space, layout, flexbox, border, buttonStyle`

### Component Props

| Name     | Type      | Description                                                                           |
| -------- | --------- | ------------------------------------------------------------------------------------- |
| children | `boolean` | Dictates the layout of an avatar component. When true it will hide the name or email. |
| icon     | `string`  | Initials of the user.                                                                 |

### Usage

```jsx live
<Button>click me</Button>
```

```jsx live
<Button variant="danger">danger</Button>
```
