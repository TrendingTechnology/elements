# Using Motion

<Status status="beta" />

## Motivation

Motion gives continuity and understanding to UIs. It helps users understand which elements relate to each other and how they can interact. Motion should be **Informative**, **Focused**, and **Expressive**.

## Timing Functions

Timing is the control of both **how long** an animation takes but also the **easing** of the animation, or the acceleration over time.

**Simple: 100ms in, 100ms out**

**Complex: 250ms in, 200ms out**
